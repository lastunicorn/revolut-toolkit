
# Revolut Bank Toolkit

A set of helpers to interact with files generated by Revolut bank.

## NuGet Package

```
Install-Package DustInTheWind.Revolut.Toolkit
```

Or via the .NET CLI:

```
dotnet add package DustInTheWind.Revolut.Toolkit
```

## Statement Files

### Step 1 - Export CSV Statement File

Go to the Salt Bank mobile application and export account statements as CSV files.

At this moment, the files should have the following header line:

`Date,Counter Party,Reference,Type,Amount (RON),Balance (RON),Spending Category,Notes`

> **Note**: The currency `RON` will be different if the export was done from another currency account.

### Step 2 - Parse the File

The `TransactionsDocument` class provides several methods to parse and load a CSV file, to accommodate different input sources. Here it is one of them:

**From a file path:**

```csharp
TransactionsDocument document = TransactionsDocument.LoadFile("transactions.csv");
```

### Step 3 - Use the Data

`TransactionsDocument` is a collection of `BankTransaction` objects:

```csharp
foreach (BankTransaction transaction in document)
    Console.WriteLine($"{transaction.Date} | {transaction.CounterParty} | {transaction.Amount}");
```

## `BankTransaction` Properties

| CSV Column       | Property        | Type                 | Description                                |
| ---------------- | --------------- | -------------------- | ------------------------------------------ |
| `Type`           | `Type`          | `TransactionType`    | The type/category of the transaction.      |
| `Product`        | `Product`       | `TransactionProduct` |                                            |
| `Started Date`   | `StartedDate`   | `DateTime`           |                                            |
| `Completed Date` | `CompletedDate` | `DateTime`           |                                            |
| `Description`    | `Description`   | `string`             |                                            |
| `Amount`         | `Amount`        | `decimal`            |                                            |
| `Fee`            | `Fee`           | `decimal`            |                                            |
| `Currency`       | `Currency`      | `Currency`           |                                            |
| `State`          | `State`         | `TransactionState`   |                                            |
| `Balance`        | `Balance`       | `decimal`            | The account balance after the transaction. |
